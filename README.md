# Stageopdracht

Automated OSINT tool by Siebe Van Rompay

This framework allows to search for company names and domains and person names and emails

## Tools

This framework uses the following Tools

- Sherlock
- Blackbird
- Spiderfoot
- OpenRefine
- Crosslinked
- Shodan
- Dehashed
- Nmap
- TheHarvester
- DNSrecon
- Whatweb
